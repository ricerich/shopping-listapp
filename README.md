# 쇼핑 리스트 앱

바닐라 HTML/CSS/JS로 만든 심플한 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 완료 토글
- 개별 아이템 삭제
- 완료 항목 일괄 삭제
- `localStorage`로 새로고침 후에도 데이터 유지
- 추가/완료 항목 통계 표시

## 사용법

`index.html` 파일을 브라우저에서 바로 열면 됩니다. 별도 서버나 빌드 과정 없음.

## 테스트

Playwright MCP를 이용한 E2E 테스트 9개 전체 통과. 상세 내역은 [`test-report.md`](test-report.md) 참고.
